# Example: Creating a Technical SOP

Here's a real-world example of documenting a hinge installation:

## Title

"Blum 170° Soft-Close Hinge Installation - Teak Upper Galley Cabinet"

## Location Context

"Upper Galley Cabinet - High-moisture environment near sink"

## Material

"Teak veneer over marine plywood core"

## Hardware Name

"Blum Clip Top BLUMOTION 170° Hinge"

## Hardware Part

"71B3550"

## Application Logic

"Customer requested soft-close hinges due to heavy sea-state conditions. The 170° opening angle provides full access in tight galley space. BLUMOTION technology prevents door slamming in rough seas. Blum hardware chosen for marine-grade corrosion resistance."

## Technique Details (Markdown)

```markdown
# Installation Steps

## Tools Required

- 35mm Forstner bit
- Cordless drill
- Phillips screwdriver
- Measuring tape
- Pencil

## Preparation

1. Mark hinge positions 3" from top and bottom of door
2. Ensure door is supported on workbench
3. Check grain direction - should run vertically

## Drilling Cup Mortise

1. Set depth stop to 12.5mm on drill
2. Position 35mm Forstner bit centered 22mm from door edge
3. Drill slowly to prevent veneer tearout
4. Clean mortise with chisel if needed

## Installing Hinge

1. Press hinge cup into mortise
2. Align mounting plate on cabinet sidewall
3. Install with #6 x 5/8" stainless screws
4. Leave 1/8" gap for adjustment

## Final Adjustment

1. Attach door and check alignment
2. Adjust using three-way adjustment screws:
   - Height: Top screw
   - Depth: Front screw
   - Side: Bottom screw
3. Test soft-close action - should engage at 30° and close smoothly

## Critical Notes

- Do NOT overtighten screws in teak veneer
- Pre-drill pilot holes in solid teak sections
- Apply teak oil to exposed edges after installation
```

## Material Constraints

"Teak veneer is only 1/16" thick over plywood core. Must pre-drill all screw holes to prevent veneer lifting. Maximum screw depth is 5/8" before hitting hollow core. Use marine-grade stainless screws only - brass will corrode in galley moisture."

## Customer Specs

"Customer wants invisible hinges with soft-close for all upper cabinets. Specified Blum brand due to warranty support. Prefers matte nickel finish to match other galley hardware."

This example shows how to document:

- ✅ **Why** this hardware was chosen (Application Logic)
- ✅ **Where** it's used (Location Context)
- ✅ **What** material considerations exist (Material + Constraints)
- ✅ **How** to install it (Step-by-step in Technique Details)
